My First Form


INTRODUCTION
------------

It is a simple form which takes user details and send a confirmation email


REQUIREMENTS
------------

No special requirements.


INSTALLATION
------------

 * Place this module in drupal/modules/custom.
 * Enable the module at drupal.com/admin/modules


USAGE
-------------
 * Go to drupal.com/first-form to view and fill in the form. 
 * You may optionally create a menu link on home page to access the form
